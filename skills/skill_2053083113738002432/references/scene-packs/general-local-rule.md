## L3 降级时的体裁感知规则（Q层叠加）

### 结构规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| GEN-S1 | 每章须有明确论点句，不以悬念开头替代立场 | 章节首段无主旨句 |
| GEN-S2 | 前言/引言须在500字内交代：读者痛点、作者主张、全书结构 | 前言超2000字仍未明确核心 |
| GEN-S3 | 各章字数差异不超过30%（均匀章节结构警报） | 某章为其他章2倍以上 |
| GEN-S4 | 附录/参考资料须独立于正文章节 | 参考夹在章节段落中 |

### 内容质量规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| GEN-C1 | 每个核心论断须有支撑（案例/数据/引用/推导之一） | "显然"、"众所周知"不加支撑 |
| GEN-C2 | 禁止连续3段使用相同句式结构（注水信号） | 三段均以"首先…其次…最后…"开头 |
| GEN-C3 | 专有名词首次出现须定义或注释 | 缩写无全称说明 |
| GEN-C4 | 数据引用须标注来源年份，不超过5年前（时效性） | "据统计"无年份 |

### 风格规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| GEN-B1 | 单段文字不超过300字（长段阅读疲劳） | 单段500字以上 |
| GEN-B2 | 同义词不在相邻3段内重复超过3次 | "赋能"在同章连续密集出现 |
| GEN-B3 | 禁止AI套话词组（详见 s5-buzzword-lexicon.json） | "在当今社会"、"随着科技发展" |
