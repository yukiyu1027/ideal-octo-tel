## L3 降级时的体裁感知规则（Q层叠加）

### 人物声音一致性规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| GHO-V1 | 全文须维持一致的人称视角（第一人称/第三人称不混用） | 段落间"我"和"他"交替出现 |
| GHO-V2 | 主体人物的口吻须与 author-meta 中采集的语言风格一致 | 作者自述"直白风格"但文稿充满书面腔 |
| GHO-V3 | 个人经历陈述须与 author-meta 中的背景信息对应 | 文中提到经历与采集信息矛盾 |
| GHO-V4 | 情感浓度须适度，避免代入性过强或过于疏离 | 连续5段均为强情绪宣泄，或全文情感零度 |

### 原创性保护规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| GHO-O1 | 核心观点须来自素材采集，不可由AI独立推导补充 | 文中出现素材库中完全没有的论断 |
| GHO-O2 | 具体数字/事件须有素材来源，不可虚构 | "我在2019年做了X"无法对应素材 |
| GHO-O3 | 主体人物的独特用语/口头禅须在全文适度保留 | 采集时有特征词但文稿中消失 |

### 交付规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| GHO-D1 | 演讲稿体裁须包含：开场钩子、核心主张（3条以内）、行动号召 | 演讲稿无明确号召结尾 |
| GHO-D2 | 专栏文体须在前150字内抛出读者痛点或争议性命题 | 专栏文章前3段均为背景铺垫 |
