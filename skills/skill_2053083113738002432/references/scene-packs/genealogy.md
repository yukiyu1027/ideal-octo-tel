## S0 初始化收集要素

写书开始前，需收集以下信息（标注★为必须）：

1. ★ 主修姓氏与郡望（如：杜氏·京兆郡）
2. ★ 修谱目的（传承记忆 / 续修 / 首修）
3. ★ 现有资料情况（是否有旧谱、族内访谈记录、地方志资料）
4. 族源地与现居住地分布
5. 预计卷数与篇幅
6. 是否需要多版本对比（旧谱 vs 新修）

## S/P/C/B 质量规则

### S 层（内容质量）

- **[MUST]** 世系关系必须前后一致，同一人物的辈分在全书各卷中不得出现矛盾
- **[MUST]** 历史事件（含战争、迁徙、科举）须有可核实来源，不得凭空编造
- **[MUST]** 人物传记中生卒年若有记载，必须通过年代合理性校验（见下方年代校验规则）
- **[SHOULD]** 迁徙路线应标注朝代年号，并与地方志相互印证
- **[SHOULD]** 各支系世系图应包含迁徙分支节点，注明分支时间和原因

### P 层（段落逻辑）

- **[MUST]** 人物传中同一人的称谓、职衔在全文保持一致
- **[MUST]** 引用族谱原文须标注原谱名称、卷次、页码
- **[SHOULD]** 同代多人传记之间不得出现年代逻辑矛盾（如兄长生年晚于弟弟）

### C 层（文字校对）

- **[MUST]** 人名、地名、官职名沿用标准历史写法，不得使用简化或口语化写法
- **[MUST]** 生卒年格式统一（如：清道光十年 / 1830年），全书格式一致
- **[SHOULD]** 避免以现代价值观评价历史人物

### B 层（全书一致性）

- **[MUST]** 各卷末提供检索索引（人名、地名、事件）
- **[SHOULD]** 全书使用统一的辈分排行用字，不得混用

## 年代合理性校验规则（评审决议 R2 加厚）

> 该规则在 P 层审校时执行，防止人物传出现生卒年逻辑错误。

```
校验条件：
1. 生年 < 卒年（卒年不早于生年）
2. 寿命范围合理（历史人物通常 20–100岁，特殊情况须注明"年寿不详"）
3. 父子代差合理（父亲生年与子女生年差距通常 15–45年）
4. 同代堂兄弟生年差距不超过 30年（超出需特别说明原因）
5. 人物所在朝代与其事迹年代匹配（如：不得出现"明朝人参加清朝科举"）
```

## 迁徙分支图规范（评审决议 R2 加厚）

世系表中涉及迁徙分支时，须在该节点标注：

```
[分支节点格式]
第N世·姓名（迁出支祖）
  ├── 迁往：省/市/镇（具体地名，尽量精确到县级）
  ├── 迁徙时间：朝代年号（如无法确定标注"约明末清初"）
  ├── 迁徙原因：（战乱/仕宦/经商/垦荒/其他）
  └── 现居后裔：（如已知，标注现居住地或联系宗亲情况）
```
