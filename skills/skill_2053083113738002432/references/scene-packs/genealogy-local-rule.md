## L3 降级时的体裁感知规则（Q层叠加）

### 年代与时序规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| GEN-Y1 | 生卒年须使用中国历史纪年+公历双标注（如"光绪十年（1884年）"） | 仅用公历年或仅用年号 |
| GEN-Y2 | 父子代际间隔须在15–40年合理范围内 | 父子年差超50年或小于12年 |
| GEN-Y3 | 同一人物的年龄/年份在多处出现时须一致 | 同一人"生于1880年"与"享年70岁"对不上 |
| GEN-Y4 | 历史事件年代须可核实（避免虚构年份） | "约清朝中期"未给范围估算 |

### 人物与关系规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| GEN-P1 | 每位传主须含：名/字（至少其一）、生卒年（或估算说明）、重要事迹 | 仅有姓名无其他信息 |
| GEN-P2 | 迁徙节点须标注：迁出地、迁入地、迁徙时间（或时代背景） | "随家族迁移"无具体信息 |
| GEN-P3 | 女性人物须使用本名（或"某氏"标注），不用"某某之妻"替代 | "王大爷之妻"无独立标注 |
| GEN-P4 | 字辈排行出现时须附全字辈序列或注明来源 | "排行第三"无字辈体系说明 |

### 史料规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| GEN-H1 | 史料引用须标注：来源类型（旧谱/地方志/口述/档案） | "据族人说"无来源类型 |
| GEN-H2 | 存疑信息须显式标注（"存疑"、"待考"） | 不确定内容作为事实陈述 |
| GEN-H3 | 多版本旧谱有分歧时须保留各版本说法，不擅自合并 | 仅采一说不注明分歧 |
