## L3 降级时的体裁感知规则（Q层叠加）

### 事实核查规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| RPT-F1 | 每个核心事实须有至少2个独立来源交叉核实 | 关键指控仅一个来源 |
| RPT-F2 | 时间线陈述须精确到日（重要节点）或月（背景事件） | "某段时期"、"近年来"无具体时间 |
| RPT-F3 | 文件/录音等证据须说明获取方式与保存状态 | "内部文件显示"无获取说明 |
| RPT-F4 | 采访来源须区分：具名来源/匿名来源/官方回应 | 所有来源混称"知情人士" |

### 当事方回应规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| RPT-R1 | 涉及当事方指控须记录"是否获得回应"（获得/拒绝回应/未能联系） | 无当事方回应记录 |
| RPT-R2 | 当事方回应须完整引用，不可断章取义 | 仅引用对报道有利的部分 |
| RPT-R3 | 匿名来源须说明匿名原因（保护隐私/职业风险等） | 匿名来源无说明 |

### 观点与事实分离规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| RPT-O1 | 记者/作者观点须与事实陈述明确区分（使用"记者认为"等标注） | 观点融入事实段落不加区分 |
| RPT-O2 | 调查结论须从事实中归纳，不超越证据 | 结论比证据支持的程度更强 |
| RPT-O3 | 标题须与正文内容相符，不夸大 | 标题措辞比正文结论更重 |
