## L3 降级时的体裁感知规则（Q层叠加）

### 个人声音规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| PB-V1 | 全书须贯穿作者个人视角，不写成第三人称自传 | "他在那段时期…"用于本人经历 |
| PB-V2 | 方法论/洞见须扎根作者亲身经历，不可泛化为通用建议 | "大家都应该…"脱离作者个人经验 |
| PB-V3 | 作者的思考过程须可见（不仅呈现结论） | 段段都是结论，无推导过程 |

### 内容真实性规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| PB-T1 | 具体事件须有时间锚点（年份/阶段），不写"某年某月" | 重要经历无时间参照 |
| PB-T2 | 对他人的描述须经当事人同意或脱敏处理（隐私保护） | 真实姓名+负面评价 |
| PB-T3 | 数据/成就陈述须可核实，避免自我美化性模糊表达 | "创造了巨大价值"无具体数字 |

### 读者价值规则

| ID | 规则 | 违反信号 |
|----|------|---------|
| PB-R1 | 每章须有读者能"带走"的洞见或方法（不只是记录往事） | 纯流水账无提炼 |
| PB-R2 | 书须回答核心命题（为什么读者应该读这本书） | 读完全书命题未被回答 |
| PB-R3 | 个人故事须与读者痛点/目标有连接（避免自嗨式叙述） | 章章讲自身成功无读者视角 |
