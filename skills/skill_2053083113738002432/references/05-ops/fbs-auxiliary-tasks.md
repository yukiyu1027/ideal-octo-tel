# 辅助模型适用任务（P2 C3）

当宿主提供**轻量/快速**模型通道时，建议将下列任务与「正文写作主模型」分离：

| 任务 | 输入 | 输出 | 说明 |
|------|------|------|------|
| 章前摘要 | 已读章节路径列表 | ≤500 字要点 | 降低主模型重复读全文 |
| 术语对齐表 | GLOSSARY + 当前章草稿 | 冲突列表 | 不直接改稿，只列问题 |
| 格式/结构检查 | 单章 Markdown | 问题清单 | 与 `quality-auditor-lite` 互补 |

未配置辅助模型时：**全部任务由主模型完成**，并遵守 `anti-stall` 与单轮文件修改上限。
