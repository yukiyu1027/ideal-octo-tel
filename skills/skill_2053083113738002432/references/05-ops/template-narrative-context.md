# 叙述 CONTEXT 冻结（模板）

> **用途**：在 **大规模成文 / 扩写 / 多章并行** 前，把「一句话大纲撑不起的细节」落成 **单文件真源**，供 team-lead、writer、质检与脚本引用。  
> **建议路径**：本书根 `.fbs/narrative-context.md`（`init-fbs-multiagent-artifacts` 可初始化空壳）。  
> **权威配合**：`author-meta.md`、`术语锁定记录.md`、`outline-freeze` / `project-config`、`.fbs/book-context-brief.md`。

## 2. 叙述参数（扩写 / 并行前必填）

| 维度 | 选择或说明 |
|------|------------|
| **篇幅节奏** | 偏案例 / 偏论述 / 均衡；章平均字数区间 |
| **案例深度** | 姓名化故事 / 匿名化 / 仅数据 |
| **风险与合规** | 必须免责声明、禁止表述、需法务复核的段落类型 |
| **对话与引语** | 是否使用直接引语、虚构对话是否允许 |

## 5. 与 GSD「discuss-phase」对齐说明
