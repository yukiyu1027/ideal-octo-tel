# 体裁化完成度清单（Warning 级）

> **用途**：在字数与 Brief 比例之外，用可勾选项降低「短而空 / 长而水」两类风险。  
> **级别**：默认 **Warning**，不单独阻断 S4；与 `chapterAcceptanceGate`、S4 勾选表并用。

---

## 通用（各体裁）

- [ ] 本章有 **一句可独立站立的结论**（非仅承上启下）。
- [ ] 关键主张下挂 **至少一类依据**：数据 / 案例 / 规范 / 可复核引用。
- [ ] 显式标出 **边界或反例**（若体裁需要：政策类 / 方法论类必选）。

---

## 白皮书 / 行业指南

- [ ] 「问题—方法—价值」链完整；读者能回答「与我何干」。
- [ ] 术语与 `术语锁定记录.md` 一致，无同义反复换名。

---

## 手册 / 培训

- [ ] 关键步骤可执行（含前提条件 / 失败时动作之一）。
- [ ] 图示或编号与正文交叉引用一致。

---

## 叙事 / 非虚构长篇

- [ ] 场景与 `story-bank.md` / 时间线无冲突。
- [ ] 人物称谓全书一致。

---

**权威质检层级**：仍以 S/P/C/B 与 [`quality-check.md`](./quality-check.md) 为准；本清单为补充。
