## 规则速查

### S1：首句具体化

**规则**：首句必须含人物/场景/数据/问题至少一个元素

**检查项**：
- [ ] 首句有明确人物（谁）
- [ ] 首句有明确场景（哪里）
- [ ] 首句有明确数据（多少）
- [ ] 首句有明确问题（什么）

**示例**：
```
✅ 首句具体：2024 年全球 AI 市场规模达到 5000 亿美元（含数据）
❌ 首句抽象：AI 市场呈现快速增长态势（无具体元素）
```

---

### S2：消灭冗余修饰

**规则**：A 类副词 ≤12 次/千字

**A 类副词**：
```
非常、十分、极其、特别、尤其、相当、比较、较为、
更为、更加、最为、极其、特别、相当、比较、较为
```

**检查项**：
- [ ] A 类副词统计次数：___
- [ ] 是否超过 12 次/千字：☐ 是 / ☐ 否

---

### S3：短句优先

**规则**：单句长度控制，避免超长句

**检查项**：
- [ ] 平均句长：___ 字
- [ ] 最长句：___ 字
- [ ] 是否有明显超长句（>50 字）：☐ 是 / ☐ 否

---

### S4：连接词降频

**规则**：C 类连接词控制频率

**C 类连接词**：
```
此外、而且、同时、另外、再者、并且、此外、还有、
不仅、而且、不但、并且、而且、同时、另外、再者
```

**检查项**：
- [ ] C 类连接词统计次数：___
- [ ] 是否过度使用同一连接词：☐ 是 / ☐ 否

---

### S5：buzzword 黑名单

**规则**：B 类 buzzword 禁用（29 个词汇）

**B 类 Buzzword**（见 `s5-buzzword-lexicon.json`）：
```
底层能力、全链路、组合拳、颗粒度、拉通、拉齐、赋能、抓手、
底座、链路、沉淀、闭环、打通、对齐、心智、势能、卡位、
卡点、矩阵、生态、中台、协同、联动、融合、整合、维度、视角、层面
```

**检查项**：
- [ ] 是否包含 B 类 buzzword：☐ 是 / ☐ 否
- [ ] 命中词汇：___

---

### S6：标点符号纪律

**规则**：顿号/冒号/感叹号使用阈值

**检查项**：
- [ ] 顿号使用是否合理（≤3 个/句）：☐ 是 / ☐ 否
- [ ] 冒号使用是否合理（≤2 个/段）：☐ 是 / ☐ 否
- [ ] 感叹号使用是否合理（≤1 个/段）：☐ 是 / ☐ 否
