> 目的：给交付、审核、维护提供单一版本口径。  
> 原则：以 **v2.1.1** 为唯一当前基线；既往版本仅保留“能力演进摘要”，不再作为运行时决策依据。

- 当前有效版本：`2.1.1`
- 当前入口基线：`SKILL.md`、`openclaw/fbs_bookwriter/SKILL.md`、`.codebuddy/agents/fbs-team-lead.md`

## 使用规则（防歧义）

- 对外说明版本时，只说：**“当前基线为 2.1.1”**。
- 讨论历史时，只引用本页归纳，不展开内部迭代流水。
- 任何脚本/文档行为冲突，以 `2.1.1` 入口文件和门禁脚本为准。
- 旧版本条目用于追溯，不可作为“跳过当前门禁”的依据。
