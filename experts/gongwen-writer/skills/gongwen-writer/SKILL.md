# 国企公文写作技能

## 触发条件
当用户提到以下关键词时自动激活本技能：
- "写材料"、"起草"、"拟稿"、"上会材料"、"公文"
- "党委会材料"、"董事会材料"、"经营层办公会"、"总经理办公会"
- "请示"、"函"、"情况说明"、"情况汇报"、"议案"
- "审查格式"、"优化用语"、"改成公文表述"

## 核心能力

### 1. 公文起草
根据用户提供的文种和关键信息，按标准框架起草公文：
- 自动匹配文种对应的标题用语和结尾用语
- 严格遵循三段式框架（背景→进展→提请审议）
- 引用前置会议纪要编号和议事规则依据

### 2. 格式审查
对已有公文进行格式合规性检查：
- 文种与结尾用语是否匹配
- 层次序数是否按"一、→（一）→1.→（1）"递进
- 附件格式是否规范
- 金额大小写是否一致

### 3. 用语优化
将口语化表述转化为规范公文用语：
- "大家注意" → "现就…事项通知如下"
- "明天" → "次日（X月X日）"
- "希望" → "请/须/务必"
- "搞定" → "完成/落实"
- "不行" → "不得/不予"

### 4. Word排版输出
按GB/T 9704标准生成Word文档：
- 标题：方正小标宋简体 二号 加粗 行距32磅
- 正文：仿宋_GB2312 三号 行距28磅
- 一级标题：黑体 三号
- 二级标题：楷体_GB2312 三号
- 页边距：上37mm 下35mm 左28mm 右26mm

## 工作流程

1. 识别用户意图：起草 / 审查 / 优化
2. 确认文种类型
3. 收集关键信息（主题、前置依据、数据、附件）
4. 按对应框架组织内容
5. 自检文种匹配和格式合规
6. 输出Word文档或优化后的文本
